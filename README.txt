Hangman Game
Jisu Park
December 2015

This is a simple hangman game using python (version 2.7).
To start the game, just type in "python hangman.py" in the command line.

RULES:
You have total 10 attempts to guess the correct word.
If your guess is wrong, you will lose one try.
If your guess is not a single alphabet character, you will lose one try.
If your guess was already guessed before, you will lose one try.
It will not matter whether you input big or small case letter, because the program will conver your input to small case letter.
If you guess the correct answer within the 10 attempts, you will win.
If you fail to guess the correct answer within the 10 attempts, you will lose.

Enjoy the game :)
